About EduManager PRO

Student Management System version 2.0.0

EduManager PRO is a complete, production-ready curriculum and student directory supervisory ledger system. Created as a modern Flask MVC implementation, it integrates user authentication pipelines, verification mail alerts, student registry CRUD panels, and search configurations.

System Capabilities
  • Secure Session Authorization: Administrators register accounts protected by secure cryptography hashes (`pbkdf2:sha256` or `scrypt`).
  • Active OTP verification: Emails are validated during registration via a six-digit one-time password module.
  • Automated Tuition Notices: Registers students and instantly delivers an automated tuition schedule mapping starting/ending days for their course program.
  • AWS Optimized Layout: Out-of-the-box configuration bindings for AWS EC2 platforms (Amazon Linux 2023) and Amazon RDS instances.
Technology Stack
Python Flask 3
MySQL Server
Bootstrap 5
Vanilla JS
